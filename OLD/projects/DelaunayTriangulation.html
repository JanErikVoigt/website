Delaunay Triangulation

I recently implemented the Delaunay Triangulation in p5js. It's animated, and therefore pretty slow, but I like it nevertheless. It is run on a set ov vertices generated by the "Fibonacci-Spiral". Obviously, this kind of data can be triangulated way easier and major optimizations are thinable. Additionally, I used a very naive approach and did not optimize in any way.
